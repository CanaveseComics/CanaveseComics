# Branch: dev

Questo è il branch principale per lo sviluppo del sito.

Caratteristiche:
- Pubblicato come anteprima tramite GitHub Pages
- Utilizzato per costruire il sito completo a partire dagli HTML preparati nel branch "client"
- Aggiornato regolarmente durante l’intero processo di sviluppo

Cosa avviene in questo branch:
- integrazione layout
- contenuti definitivi
- CSS, JS, immagini e componenti del sito
- test di visualizzazione tramite anteprima GitHub Pages
- verifiche responsive e prestazionali

Workflow:
1. Il cliente fornisce gli HTML iniziali nel branch "client".
2. Tutto lo sviluppo completo avviene qui su "dev".
3. Una volta approvato, il codice finale viene mergiato nel branch "main".

⚠️ Nota:
Questo branch NON è collegato al dominio ufficiale.
Serve esclusivamente come ambiente di sviluppo e anteprima.

❗ IMPORTANTE — NON FARE PULL REQUEST VERSO MAIN: ignorare eventuali messaggi di GitHub del tipo “Compare & pull request”.
Sono suggerimenti automatici di GitHub e non devono essere utilizzati
durante la fase attuale del progetto.
